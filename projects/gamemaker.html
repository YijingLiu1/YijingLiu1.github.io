Post Mortem

This 4-week game/game maker project was based on a classic game 'Tower of the Sorcerer'. Beyond built-in game levels, with the game maker, users can build their own game with over 50 different built-in items/enemies/NPCs. More features can still be done for future development.

Gameplay

Battles in the game uses turn-based logic. The player and enemy would take turns to attack each other until one side down, with damage based on atk/def stats. More battle rules could be introduced in the gameplay, such as magic (ignore defense), preemptive (enemy attack first), debuff.

Features

Current gameplay binds NPC with fixed dialog and no actual feature to interact with player. Besides, store is not added within the game, so the player could only boost the hero by items and equipments. More diverse features, like NPC and store, could be added to improve the gameplay.

Level Switch

Current level switch uses a teleport mechanism, set with one upstairs case to one downstairs case, whereas the map design could be added with more stairs to increase the maze challenge. Related feature could be implemented in future development.

Stat Editing

Some sprites(item/enemy) have stats editable (atk/def/hp). But for player/NPC this is not provided. The initial stats of player could be edited to increase the freedom of level design. It may also be fun to provide dialog editing in this section for NPCs.
