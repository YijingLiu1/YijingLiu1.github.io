Breakout! - An Arcade Game

Originally Developed and Published by Atari, Inc.

Key Features

1.5 Levels Built-in
2.Frame Capped to 60 FPS
3.English/Chinese Support
4.Levels Editable in Text
5.'Space' for Start
6.'←' '→' for Move
7.'Esc' for Pause
8.'H' for Increasing Lives
9.'N' for Next Level

Post Mortem

This game project was built in 2 weeks, implemented with all required features. However, more features can be done to increase the game performance and user experience.

Gameplay

Game now starts with Stage 1. However, starting with a main menu was planned to make the system more consistent. Also, bricks could be implemented with different stiffness, which means different types of bricks would be able to undertake different numbers of hits.

More user-friendly features could be added, including having a score board to save players' highest score, a menu item to select different levels, also the ability to switch language directly in game instead of the config file.

Visual & Auditory Experience

Game was rendered with standard Breakout graphics and sound effects. However, as first planned, the boundary of the level would be rendered as "⛩", same theme with added BGM.

The graphics would also have stone lanterns on both sides and a background. Sound effects would be changed into more theme-related effects.
